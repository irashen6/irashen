# IRASHEN - QQ堂风格主页

一个采用QQ堂游戏风格设计的个人主页，具有动态动画和交互效果。

## 🎮 功能特性

- QQ堂经典视觉风格
- 角色移动控制（方向键或WASD）
- 放置泡泡功能（空格键）
- 道具收集系统
- 粒子特效
- 响应式设计

## 🕹️ 操作说明

- **↑↓←→ / WASD** - 移动角色
- **空格键** - 放置泡泡
- **开始游戏** - 开始交互
- **重置** - 重置游戏状态

## 🚀 使用方法

直接在浏览器中打开 `index.html` 即可查看效果。
